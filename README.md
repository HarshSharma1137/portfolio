# 🚀 Harsh Sharma — Portfolio Website

A production-grade personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## 🌐 Live Demo
[portfolio-nine-peach-73.vercel.app](https://portfolio-nine-peach-73.vercel.app)

## 🛠 Tech Stack
- **Framework:** Next.js 16 + TypeScript
- **Styling:** Tailwind CSS + Inline Styles
- **Fonts:** Sora + DM Sans + DM Mono
- **Deployment:** Vercel

## ✨ Features
- Dark / Light mode toggle
- Smooth scroll + scroll reveal animations
- Sticky navbar with active section highlight
- Fully responsive + mobile first
- Sections: Hero, About, Skills, Projects, Education, Contact

## 📁 Projects Showcased
- **Web Defend** — Web Application Firewall (WAF)
- **College Discovery Platform** — Full-stack college search app
- **Async Document Processing System** — FastAPI based async processor
